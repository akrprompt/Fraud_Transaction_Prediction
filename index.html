Fraud Transaction Prediction

โมเดลทำนายการฉ้อโกงในธุรกรรมทางการเงิน

Project Overview

โปรเจคนี้มีวัตถุประสงค์เพื่อวิเคราะห์และทำนายธุรกรรมที่เป็นการฉ้อโกง โดยใช้เทคนิค Machine Learning

ใช้ข้อมูลธุรกรรมทางการเงิน ประกอบด้วยคอลัมน์ต่างๆ เช่น ประเภทธุรกรรม จำนวนเงิน และยอดคงเหลือ

เทคโนโลยีที่ใช้
  • Python
  • Pandas & NumPy
  • Scikit-learn
  • Matplotlib & Seaborn
  • Imbalanced-learn

Data Exploration

ข้อมูลประกอบด้วยธุรกรรมทางการเงินต่างๆ โดยมีข้อมูลทั้งธุรกรรมปกติและธุรกรรมที่เป็นการฉ้อโกง

ตัวอย่างข้อมูล

step    type    amount    nameOrig    oldbalanceOrg    newbalanceOrig    nameDest    oldbalanceDest    newbalanceDest    isFraud

ลักษณะของข้อมูล

  • step: ขั้นตอนเวลาของธุรกรรม (1 หน่วย = 1 ชั่วโมง)
  • type: ประเภทของธุรกรรม (PAYMENT, TRANSFER, CASH_OUT, DEBIT)
  • amount: จำนวนเงินในธุรกรรม
  • nameOrig: รหัสลูกค้าผู้โอน
  • nameDest: รหัสลูกค้าผู้รับโอน
  • isFraud: ป้ายกำกับว่าเป็นการฉ้อโกงหรือไม่ (1 = ฉ้อโกง, 0 = ไม่ฉ้อโกง)

Modeling Approach

เราใช้เทคนิคต่อไปนี้ในการสร้างโมเดลทำนายการฉ้อโกง:

Data Preprocessing

  • การจัดการค่าที่หายไป
  • Feature Engineering
  • การแปลงข้อมูล Categorical
  • การแก้ปัญหา Imbalanced Data ด้วย SMOTE

Algorithms

  • Logistic Regression
  • Random Forest
  • XGBoost
  • การปรับแต่ง Hyperparameters

ตัวอย่างโค้ด Feature Engineering

# สร้าง features ใหม่
df['typeTransfer'] = np.where(df.type == 'TRANSFER', 1, 0)
df['typeCashOut'] = np.where(df.type == 'CASH_OUT', 1, 0)
df['typePayment'] = np.where(df.type == 'PAYMENT', 1, 0)
df['typeDebit'] = np.where(df.type == 'DEBIT', 1, 0)

# สร้าง features จากความแตกต่างของยอดเงิน
df['diffOrig'] = df.newbalanceOrig - df.oldbalanceOrg
df['diffDest'] = df.newbalanceDest - df.oldbalanceDest

Results & Evaluation

Model Performance

โมเดล XGBoost ให้ผลลัพธ์ดีที่สุด:

  • Accuracy: 0.998
  • Precision: 0.942
  • Recall: 0.879
  • F1-Score: 0.909
  • AUC-ROC: 0.990

Feature Importance

Conclusion

โปรเจคนี้แสดงให้เห็นว่าสามารถใช้ Machine Learning ในการตรวจจับการฉ้อโกงในธุรกรรมทางการเงินได้อย่างมีประสิทธิภาพ

เทคนิค SMOTE ช่วยแก้ปัญหา Imbalanced Data ได้ดี และโมเดล XGBoost ให้ผลลัพธ์ที่ดีที่สุด

  • ปรับใช้ในระบบ Real-time Fraud Detection
